# BLS 项目简介与使用说明

## 项目简介

BLS 是一个前后端分离的企业级管理系统，前端基于 Ant Design Pro + React + Umi Max 构建，后端采用 Koa + TypeScript + MySQL 实现。项目支持 JWT 鉴权、RBAC 权限控制、多租户隔离以及统一的接口返回规范，适合中后台业务快速开发与扩展。

## 系统特点

- 前后端分离，接口清晰，便于协作开发
- 支持登录鉴权与用户权限控制
- 支持多租户数据隔离，满足 SaaS 场景
- 后端采用分层架构，便于维护与复用
- 前端基于 Ant Design Pro，界面规范、易于二次开发

## 使用说明

### 1. 启动后端

```bash
cd bls-server
npm install
copy .env.example .env
npm run db:init
npm run dev
```

默认管理员账号：`admin`
密码：`admin123`

### 2. 启动前端

```bash
cd bls-admin
npm install
npm run start
```

### 3. 访问系统

前端启动后，打开浏览器访问本地开发地址即可使用系统。登录后可进行用户、角色、菜单及租户等模块管理。

## 适用场景

适用于企业后台管理系统、SaaS 平台、权限管理平台及需要快速搭建标准化中后台应用的项目。
