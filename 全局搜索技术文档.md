# 全局搜索技术文档

基于 MySQL 的自建搜索引擎，采用**配置驱动 + 倒排索引重建**模式。

## 1. 核心数据模型

### 搜索配置表 `sys_global_search_config`

定义哪些业务表参与搜索及字段映射：

| 字段 | 说明 |
|------|------|
| `module_key` | 模块标识 |
| `source_table` | 源表名 |
| `biz_id_field` | 业务主键字段 |
| `title_field` / `subtitle_field` / `content_fields` | 标题/副标题/内容字段映射 |
| `route_path` | 前端跳转路径 |
| `permission` | 搜索权限标识 |
| `tenant_field` / `deleted_field` / `status_field` | 过滤字段 |
| `enabled` | 是否启用 |

### 搜索索引表 `sys_search_index`

扁平化索引存储，`index_id = tenantId:moduleKey:bizId`：

| 字段 | 说明 |
|------|------|
| `index_id` | 组合主键 |
| `tenant_id` / `module_key` / `biz_id` | 租户/模块/业务ID |
| `title` / `subtitle` / `content` | 搜索内容（LIKE 模糊匹配） |
| `permission` | 权限过滤 |
| `status` / `deleted` | 状态过滤 |

建有 9 个索引覆盖常见查询场景。

## 2. 索引写入

### 实时增量同步

业务表变更时调用对应同步函数，通过 `REPLACE INTO` 写入/更新索引。

### 手动重建

```
POST /system/global-search/index/rebuild
```

1. 读取 `enabled=1` 的配置
2. 动态拼接源表查询 → 清空旧索引 → 批量 REPLACE INTO
3. 字段名校验防 SQL 注入（仅允许字母/数字/下划线）

## 3. API 端点

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | `/system/global-search/search?keyword=xx` | `system:global-search:search` | 核心搜索 |
| GET | `/system/global-search/config/list` | `system:global-search:config:list` | 配置列表 |
| POST | `/system/global-search/config/save` | `system:global-search:config:save` | 新增/更新配置 |
| DELETE | `/system/global-search/config/:id` | `system:global-search:config:delete` | 逻辑删除配置 |
| GET | `/system/global-search/index/modules` | `system:search-index:rebuild` | 已启用模块列表 |
| POST | `/system/global-search/index/rebuild` | `system:search-index:rebuild` | 索引重建 |

## 4. 搜索逻辑

```
GET /system/global-search/search?keyword=xx
```

1. keyword ≥ 2 字符才执行
2. 按 `tenant_id` 隔离，`deleted=0`，`status='0'`
3. `title/subtitle/content` LIKE `%keyword%`
4. 按 `create_time DESC` → 限 50 条
5. 按 `module_key` 分组返回 `{moduleKey, moduleName, routePath, list: [{id, title, subtitle, ...}]}`

## 5. 接入新业务表

1. `sys_global_search_config` 新增启用配置
2. 业务增/改/删时调用同步索引函数
3. 历史数据通过 `/index/rebuild` 重建
4. 补齐权限码定义

## 6. Redis 缓存

| Key | TTL | 说明 |
|-----|-----|------|
| `global_search:configs` | 30min | 启用配置列表 |
| `global_search:permissions:{tenantId}:{userId}` | 10min | 用户权限 |
| `global_search:roles:{tenantId}:{userId}` | 10min | 用户角色 |
